# Nextron - Ecommerce de Eletrônicos 🚀

Bem-vindo ao repositório do projeto **Nextron**, uma landing page moderna e responsiva desenvolvida para um e-commerce de eletrônicos. Este projeto foi construído com foco em organização de código, design limpo e experiência do usuário.

## 📋 Sobre o Projeto

O Nextron é uma vitrine virtual para produtos de tecnologia, como smartphones, notebooks, smart TVs e acessórios. O objetivo principal é apresentar os produtos de forma atraente, facilitando a navegação do usuário através de categorias e destaques.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada utilizando:
  - **Variáveis CSS**: Para facilitar a manutenção de cores e fontes.
  - **Flexbox & Grid**: Para criação de layouts responsivos e alinhamentos precisos.
  - **Componentização**: Organização dos estilos em arquivos separados por componentes (Header, Footer, Hero, etc.).
- **Fontes Google**: Utilização da fonte 'Ubuntu' para uma leitura moderna e clara.

---

## 📂 Estrutura de Pastas

A organização do projeto segue boas práticas de desenvolvimento front-end:

```text
ecommerce-eletronicos/
├── css/
│   ├── componentes/        # Estilos específicos para cada parte do site
│   │   ├── footer.css
│   │   ├── header.css
│   │   ├── hero.css
│   │   ├── product-card.css
│   │   └── product-grip.css
│   ├── reset.css           # Limpeza de estilos padrões do navegador
│   ├── style.css           # Estilos globais e principais
│   └── variables.css       # Definição de cores e fontes
├── images/
│   ├── banners/            # Imagens de fundo e promocionais
│   ├── icons/              # Ícones SVG (redes sociais, carrinho, menu)
│   ├── logo/               # Logotipo da marca
│   └── products/           # Fotos dos produtos (celulares, fones, etc.)
└── index.html              # Arquivo principal do projeto
```

---

## ✨ Funcionalidades

- **Design Responsivo**: O site se adapta a diferentes tamanhos de tela (desktop, tablet e mobile).
- **Menu de Navegação**: Acesso rápido às seções de Início, Categorias, Ofertas e Contato.
- **Seção Hero**: Destaque principal com chamadas para ação (CTA).
- **Categorias**: Cards interativos para filtrar produtos por tipo.
- **Grade de Produtos**: Layout organizado para exibição dos produtos em destaque.
- **Rodapé Completo**: Links informativos, inscrição em newsletter e ícones de redes sociais.

## 🚀 Como Executar o Projeto

Para visualizar o projeto localmente, siga os passos abaixo:

1. Faça o download ou clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/ecommerce-nextron.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd ecommerce-nextron/ecommerce-eletronicos
   ```
3. Abra o arquivo `index.html` em seu navegador de preferência (Google Chrome, Firefox, Edge, etc.).
